                    LUZ - APP DE BIENESTAR INTERACTIVA
                  Explicación Simple para Presentación
===============================================================================

1. ¿QUÉ ES LUZ?
===============================================================================

Luz es una aplicación móvil que ayuda a las personas a mejorar su bienestar 
emocional usando Inteligencia Artificial. La app aprende de cada usuario para 
dar recomendaciones personalizadas de actividades que mejoren su estado de ánimo.

CONCEPTOS CLAVE:
• Natural Chemicals: Actividades que estimulan químicos naturales del cerebro
• IA que aprende: Algoritmos que mejoran con cada uso
• Feedback personalizado: Sugerencias únicas para cada persona


2. MODELO DE APRENDIZAJE - CÓMO APRENDE LA APP
===============================================================================

La app usa 5 algoritmos de Inteligencia Artificial que trabajan juntos:

A) RANDOM FOREST - "El Clasificador"
------------------------------------
¿Qué hace? Analiza el estado emocional actual

DATOS QUE RECOGE:
• Estados emocionales: Felicidad, estrés, motivación (valores 0.0-1.0)
• Hora del día: Mañana, tarde, noche
• Día de la semana: Lunes, martes, etc.
• Historial de 10 interacciones recientes
• Contexto situacional: Trabajo, casa, social

COMPONENTES DEL ALGORITMO:
| Parte              | Función                                        |
| ------------------ | ---------------------------------------------- |
| Entrada            | Vector 3D [felicidad, estrés, motivación]     |
| Múltiples Árboles  | 100 árboles que votan independientemente      |
| División de Datos  | Cada árbol entrena con muestra diferente       |
| Votación           | Combina predicciones de todos los árboles      |
| Resultado          | Clasificación emocional (0: muy bajo → 4: muy alto) |

Ejemplo práctico:
   Usuario dice: Felicidad=70%, Estrés=30%, Motivación=80%
   → La IA clasifica: "Estado BUENO pero con algo de estrés"
   → Resultado: Sugiere actividades para reducir estrés

B) RED NEURONAL AUTOENCODER - "El Detective de Patrones"
--------------------------------------------------------
¿Qué hace? Encuentra patrones emocionales únicos de cada persona

DATOS QUE RECOGE:
• Combinaciones emocionales complejas: Patrones de 3 emociones simultáneas
• Variabilidad emocional: Qué tan estables o fluctuantes son las emociones
• Correlaciones ocultas: Relaciones entre emociones que no son obvias
• Embeddings latentes: Representación matemática de 4 dimensiones
• Historial emocional: Todos los estados previos del usuario

COMPONENTES DEL ALGORITMO:
| Parte     | Función                                      |
| --------- | -------------------------------------------- |
| Entrada   | Estados emocionales numéricos [3 valores]    |
| Encoder   | Aprende relaciones entre emociones           |
| Embedding | Representación compacta del estado emocional |
| Decoder   | Garantiza que el embedding sea informativo   |
| Resultado | Vector latente listo para clustering y RL    |

Ejemplo práctico:
   María siempre tiene: Alta felicidad + Bajo estrés + Alta motivación
   Carlos siempre tiene: Media felicidad + Alto estrés + Baja motivación
   → La IA descubre que María es "Tipo Equilibrado"
   → La IA descubre que Carlos es "Tipo Presionado"
   → Da diferentes sugerencias a cada uno

C) Q-LEARNING - "El Que Recuerda lo Que Funciona"
-------------------------------------------------
¿Qué hace? Aprende qué actividades funcionan mejor para cada persona

DATOS QUE RECOGE:
• Actividades realizadas: Tipo de Natural Chemical, intensidad (1-5)
• Feedback del usuario: Calificación de efectividad (1-5 estrellas)
• Estado emocional previo: Antes de hacer la actividad
• Estado emocional posterior: Después de hacer la actividad
• Tiempo invertido: Duración de cada actividad
• Contexto temporal: Cuándo fue más efectiva cada actividad
• Preferencias personales: Qué funciona mejor para cada persona

COMPONENTES DEL ALGORITMO:
| Parte          | Función                                           |
| -------------- | ------------------------------------------------- |
| Estado         | Discretización emocional (27 estados posibles)   |
| Acción         | 4 Natural Chemicals + variaciones intensidad     |
| Q-Table        | Matriz que almacena valor de cada estado-acción  |
| Exploración    | ε=0.2 prueba cosas nuevas                        |
| Resultado      | Acción óptima basada en experiencia acumulada    |

Ejemplo práctico:
   Juan probó "Meditation" 5 veces → le dio 4-5 estrellas siempre
   Juan probó "Exercise" 3 veces → le dio 2-3 estrellas siempre
   → La IA aprende: "Para Juan, sugiere más Meditation que Exercise"

D) NLP (ANÁLISIS DE TEXTO) - "El Que Entiende Sentimientos"
------------------------------------------------------------
¿Qué hace? Analiza lo que escribe el usuario

DATOS QUE RECOGE:
• Notas personales: Texto libre que escribe el usuario al hacer actividades
• Emociones liberadas: Textos del Alma Board sobre emociones tóxicas
• Gratitudes: Textos de agradecimiento y pensamientos positivos
• Palabras clave emocionales: Detección de sentimientos en el texto
• Contexto semántico: Significado profundo de lo que expresa el usuario
• Patrones de lenguaje: Cómo se expresa cada persona cuando está bien/mal

COMPONENTES DEL ALGORITMO:
| Parte              | Función                                        |
| ------------------ | ---------------------------------------------- |
| Entrada            | Texto en lenguaje natural (español/inglés)    |
| Tokenización       | Divide texto en palabras y símbolos           |
| Transformer        | Modelo pre-entrenado que entiende contexto    |
| Embedding Semántico| Vector de 384 dimensiones del significado     |
| Resultado          | Análisis sentimientos + frases personalizadas |

Ejemplo práctico:
   Usuario escribe: "Me siento abrumado por el trabajo"
   → La IA detecta: Estrés laboral, necesidad de calma
   → Genera frase motivadora: "Es normal sentirse así, respira profundo"

E) CLUSTERING - "El Que Encuentra Gemelos Emocionales"
------------------------------------------------------
¿Qué hace? Encuentra usuarios con patrones similares

DATOS QUE RECOGE:
• Perfiles emocionales: Embeddings de 4 dimensiones de cada usuario
• Patrones de uso: Qué actividades prefiere cada persona
• Efectividad por grupos: Qué funciona mejor para cada tipo de usuario
• Características demográficas: Edad, género, ubicación (opcional)
• Horarios de actividad: Cuándo usa más la app cada grupo
• Evolución temporal: Cómo cambian los patrones de cada grupo

COMPONENTES DEL ALGORITMO:
| Parte              | Función                                        |
| ------------------ | ---------------------------------------------- |
| Entrada            | Embeddings latentes de 4D del autoencoder     |
| Inicialización     | K-means++ coloca centroides inteligentemente  |
| Asignación         | Cada usuario se asigna al cluster más cercano |
| Actualización      | Centroides se mueven al centro de cada grupo  |
| Resultado          | 5 arquetipos emocionales con características   |

Ejemplo práctico:
   Ana y Luis tienen patrones emocionales similares
   Ana probó "Gratitud" y le funcionó muy bien
   → La IA sugiere "Gratitud" a Luis también


3. GUARDADO Y PROCESAMIENTO DE DATOS
===============================================================================

La app guarda información en 11 tablas diferentes:

DATOS QUE SE GUARDAN:
• Estados emocionales: Felicidad, estrés, motivación cada vez que usa la app
• Actividades realizadas: Qué Natural Chemical hizo, con qué intensidad
• Feedback del usuario: Qué tan útil fue cada actividad (1-5 estrellas)
• Notas personales: Lo que escribe el usuario
• Patrones de uso: Cuándo usa la app, qué hace más seguido

EJEMPLO DE DATOS REALES:
┌─────────────────────────────────────────────────────────────────┐
│ Usuario: María González                                         │
│ Fecha: 31/01/2026 10:30                                       │
│ Estado previo: Felicidad=60%, Estrés=70%, Motivación=40%      │
│ Actividad: Liberó Dopamina con intensidad 4/5                 │
│ Nota: "Me sentía sin energía para trabajar"                   │
│ Estado después: Felicidad=75%, Estrés=50%, Motivación=85%     │
│ Feedback: 5 estrellas - "Me ayudó muchísimo"                  │
└─────────────────────────────────────────────────────────────────┘

PROCESAMIENTO INTELIGENTE:
• La IA analiza estos datos en tiempo real
• Aprende que para María, la Dopamina funciona muy bien cuando está desmotivada
• Actualiza las recomendaciones automáticamente
• La próxima vez que María tenga baja motivación, sugerirá Dopamina primero


4. NATURAL CHEMICALS - LAS ACTIVIDADES INTELIGENTES
===============================================================================

La app sugiere 4 tipos de "químicos naturales" basándose en neurociencia:

SEROTONINA - "Químico de la Felicidad"
• Actividades: Caminar al sol, meditar, música relajante
• Efecto: +15% felicidad, -10% estrés
• Se sugiere cuando: Baja felicidad o alto estrés

DOPAMINA - "Químico de la Motivación"
• Actividades: Completar tareas, ejercicio, nuevas experiencias
• Efecto: +20% motivación, +10% felicidad
• Se sugiere cuando: Baja motivación o falta de energía

ENDORFINAS - "Analgésicos Naturales"
• Actividades: Ejercicio intenso, risa, chocolate negro
• Efecto: +12% felicidad, -15% estrés
• Se sugiere cuando: Alto estrés o dolor emocional

OXITOCINA - "Químico del Amor"
• Actividades: Abrazar, tiempo con familia, actos de bondad
• Efecto: +18% felicidad, +10% motivación
• Se sugiere cuando: Soledad o necesidad de conexión


5. FLUJO COMPLETO CON EJEMPLO REAL
===============================================================================

Veamos cómo funciona la app paso a paso con un usuario real:

PASO 1 - USUARIO INICIA SESIÓN
Pedro abre la app un lunes por la mañana
Estado emocional: Felicidad=40%, Estrés=80%, Motivación=30%

PASO 2 - ANÁLISIS INTELIGENTE (en 0.2 segundos)
• Random Forest analiza: "Estado PREOCUPANTE - alto estrés, baja felicidad"
• Autoencoder encuentra el patrón: "Tipo estrés laboral lunes por la mañana"
• Q-Learning revisa historial: "A Pedro le funciona bien Serotonina y Endorfinas"
• Clustering encuentra: "Usuarios como Pedro responden bien a actividades de calma"

PASO 3 - SUGERENCIAS PERSONALIZADAS
La app sugiere:
1. ENDORFINAS (prioridad alta) - "Para reducir tu estrés"
   • Ejercicio suave 10 minutos
   • Respiración profunda
   • Escuchar música que te guste
   
2. SEROTONINA (prioridad media) - "Para mejorar tu ánimo"
   • Caminar 5 minutos al aire libre
   • Meditar 3 minutos
   • Llamar a un amigo

PASO 4 - PEDRO ELIGE Y ACTÚA
Pedro selecciona: "Ejercicio suave 10 minutos" (Endorfinas)
Intensidad elegida: 3/5
Nota: "Necesito algo rápido antes de la reunión"

PASO 5 - ACTUALIZACIÓN AUTOMÁTICA
La app calcula automáticamente:
• Nuevo estado: Felicidad=55%, Estrés=60%, Motivación=65%
• Las burbujas emocionales se actualizan en tiempo real
• Se muestra progreso: "¡Mejoraste tu estrés en 20%!"

PASO 6 - FEEDBACK Y APRENDIZAJE
Pedro califica la actividad: 4/5 estrellas
La IA aprende:
• "Ejercicio suave funciona bien para Pedro cuando tiene estrés laboral"
• "Los lunes por la mañana, Pedro prefiere actividades rápidas"
• Actualiza la Q-Table para futuras recomendaciones

PASO 7 - MEJORA CONTINUA
La próxima vez que Pedro tenga un perfil similar:
• La app sugerirá "Ejercicio suave" con prioridad más alta
• Ajustará las recomendaciones basándose en lo que funcionó
• Generará frases motivadoras personalizadas para Pedro


6. VENTAJAS DEL MODELO DE APRENDIZAJE
===============================================================================

PERSONALIZACIÓN REAL:
• Cada usuario recibe recomendaciones únicas
• Mejora automáticamente con cada uso
• No da consejos genéricos, sino específicos para cada persona

APRENDIZAJE CONTINUO:
• Los algoritmos se actualizan en tiempo real
• Aprende de errores y aciertos
• Se adapta a cambios en los patrones del usuario

RESPALDO CIENTÍFICO:
• Basado en neurociencia real (serotonina, dopamina, etc.)
• Algoritmos probados de Machine Learning
• Datos respaldados por investigación

TRANSPARENCIA:
• El usuario siempre sabe si está online (IA completa) u offline (algoritmos básicos)
• Puede ver cómo mejoran sus métricas emocionales
• Datos archivados de forma segura para investigación


7. EJEMPLO DE EVOLUCIÓN DE UN USUARIO
===============================================================================

USUARIO: SOFÍA - EVOLUCIÓN EN 30 DÍAS

DÍA 1:
Estado inicial: Felicidad=50%, Estrés=70%, Motivación=40%
La app no conoce a Sofía, da sugerencias generales
Sofía prueba Serotonina (caminar), califica 3/5

DÍA 7:
La app ya detectó que Sofía prefiere actividades sociales
Estado: Felicidad=60%, Estrés=60%, Motivación=55%
Sugiere Oxitocina (llamar amigos), Sofía califica 5/5

DÍA 15:
Patrón claro: Sofía es "Tipo Social", necesita conexión humana
Estado: Felicidad=70%, Estrés=50%, Motivación=70%
IA sugiere automáticamente actividades sociales primero

DÍA 30:
Estado promedio: Felicidad=80%, Estrés=30%, Motivación=85%
Sofía ha mejorado 30 puntos en felicidad
La app mantiene su bienestar con sugerencias precisas

RESULTADO: La IA aprendió el perfil único de Sofía y la ayudó a mejorar 
significativamente su bienestar emocional.


8. SEGURIDAD Y PRIVACIDAD DE DATOS
===============================================================================

PROTECCIÓN DE DATOS:
• Datos encriptados y seguros
• Usuario controla qué información comparte
• Opción de usar la app offline (sin enviar datos al servidor)

LIMPIEZA AUTOMÁTICA:
• Cada mes se eliminan datos obsoletos automáticamente
• Datos importantes se archivan para investigación (anonimizados)
• Usuario puede borrar su información cuando quiera

MODO OFFLINE:
• La app funciona sin conexión a internet
• Usa algoritmos básicos locales
• Los datos quedan en el dispositivo del usuario
